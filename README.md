## WhisperBox-BE

### Introduction

WhisperBox is an anonymous messaging platform that allows users to create a unique public link and receive messages without requiring authentication.

### Setup

Clone the repository to you local machine

```
git clone https://github.com/Lex043/WhisperBox-Backend.git
```

Ensure that you have Typescript and Postgresql installed on your machine

```
cd WhisperBox-BE
```

Run the following command to install the neccessary dependencies

```
npm install
```

Run the following command to start the server

```
npm run dev
```

The server will run on http://localhost:8000 by default
